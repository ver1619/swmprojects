# Password Strength Checking Program 

**This is a simple GO Program used to check Password strength**
---
+ User Input
+ Verfies whether the input password has
    - a Uppercase (A - Z)
    - a Lowercase (a - z)
    - a Digit (0-9)
    - a Special character (@,!,#,$)
    - Password length is >= 8
+ Distinguish password strength in Weak, Medium, Strong
